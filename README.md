# Toufique Zaman Topu - Personal Portfolio & Blog

## 👨‍💻 About
This is the personal portfolio and blog repository for **Toufique Zaman Topu**, a Cybersecurity Researcher, Penetration Tester, Bug Bounty Hunter, and the Founder of Bug Mohol. This platform is designed to showcase security research, vulnerability assessments, professional milestones, and technical write-ups.

## ✨ Features
* **Interactive UI/UX**: Features a terminal-style preloader and typing animations.
* **Responsive Design**: Fully responsive layout built with custom CSS and Bootstrap 5.
* **Dynamic Blog System**: Custom HTML/JS-based blogging system with category tags, SEO optimization, and dynamic article rendering.
* **Portfolio & Certifications**: Modal-based certificate viewer for achievements.
* **Contact Integration**: Direct message functionality using Formsubmit.

## 🛠️ Tech Stack
* **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
* **Framework**: Bootstrap 5.3
* **Libraries**: Typed.js for dynamic text animations
* **Hosting**: GitHub Pages

## 📁 Repository Structure
* `index.html`: Main portfolio page containing Hero, About, Experience, Achievements, and Contact sections.
* `blog.html`: The central hub for all blog articles and write-ups.
* `blog/`: Directory containing all individual full blog posts.
* `assets/`: Contains CSS, JS scripts, images, and blog data.
* `BLOG-INSTRUCTIONS.md`: Detailed guidelines on how to publish new content.

## 📝 How to Add a New Blog Post
Creating a new blog post is simple and structured:
* Duplicate `NEW-POST-TEMPLATE.html` and save it inside the `blog/` folder with a proper URL-friendly name.
* Fill in the SEO title, 160-character description, and relevant keywords.
* Choose a category from `ctf` (Red), `bugbounty` (Blue), `tutorial` (Yellow), or `news` (Purple).
* Open `blog.html` and add the link to your new file inside the blog card section.
